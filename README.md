A beginner-friendly MongoDB practice repository containing essential CRUD operations, query operators, and collection management examples.

## 📚 Topics Covered

### 🗄️ Database & Collection
- Create Database
- Switch Database
- Create Collection
- Show Databases
- Show Collections

### ➕ Insert Operations
- insertOne()
- insertMany()

### 🔍 Read Operations
- find()
- findOne()
- Query Specific Documents

### 🎯 Query Operators
- Equality Operator
- Greater Than ($gt)
- Greater Than or Equal ($gte)
- Less Than ($lt)
- Less Than or Equal ($lte)
- Not Equal ($ne)
- In Operator ($in)
- Not In Operator ($nin)

### 🔗 Logical Operators
- AND
- OR
- NOT
- NOR

### ✏️ Update Operations
- updateOne()
- updateMany()
- $set Operator

### ❌ Delete Operations
- deleteOne()
- deleteMany()

### 📊 Sorting & Limiting
- sort()
- limit()
- skip()

### 📈 Document Operations
- countDocuments()
- Pretty Print Results

## 🛠️ Technologies Used

- MongoDB
- MongoDB Compass
- Mongo Shell (mongosh)
- Visual Studio Code
